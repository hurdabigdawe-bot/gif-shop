# 🎮 GIF Shop Gaming Platform

Modern webes gaming platform HTML, CSS, JavaScript és Firebase használatával.

## 🚀 Funkciók

### 🃏 Blackjack Casino
- Blackjack 3:2 kifizetés
- Double Down
- Dealer Soft 17 szabály
- Achievement rendszer
- Kredit rendszer
- Win / Loss statisztika
- Rang rendszer

### 🎰 Slot Machine
- Kredit alapú játék
- Nyeremény számítás
- Animációk

### 🐦 Flappy Game
- Pontszám mentés
- Toplista

### 👤 Profil rendszer
- Firebase Authentication
- Felhasználói profil
- Kredit egyenleg
- Rangok
- Achievementek

### 🏆 Toplista
- Játékos rangsor
- Győzelmek
- Kreditek
